# Kiponnect

A simple and powerful mobile app for **KIPO College** students.

## Features
- **Social Feed:** Post text with optional photos.
- **Schedule:** View your college classes.
- **Profile:** Edit your name and avatar.
- **Languages:** Switch between English and Russian.
- **Offline Mode:** All data is saved locally on your device.

## Quick Start
1. Run `flutter pub get` to install packages.
2. Run `flutter run` to start the app.

## Tech Stack
- **Flutter** (Material 3)
- **Provider** (State Management)
- **Shared Preferences** (Local Storage)
